# ₿ BTC Advanced Dashboard

A modern, interactive Bitcoin analytics dashboard built with **Streamlit** and **Python**.
This project provides real-time market insights, portfolio analysis, and scenario-based simulations in a clean, professional interface.

---

## 🚀 Live Demo

👉 [BTC Advanced Dashboard](https://btc-advanced-dashboard-jw5qfxa3jkcxdxnyx8bwqn.streamlit.app)

---

## 📊 Features

* 📈 **Live BTC Price Tracking**
  Real-time Bitcoin price, market cap, and volume data.

* 🕯 **Candlestick Chart (OHLC)**
  Interactive price chart with moving averages (MA7, MA30).

* 🎯 **Price Simulator**
  Test different BTC price scenarios and instantly see profit/loss impact.

* 📉 **Profit & Loss Analysis**
  Automatic calculation of portfolio value, P/L, and percentage returns.

* 🔥 **Profit Heatmap**
  Visualize profit/loss across multiple price scenarios.

* ⚖️ **Break-even Analysis**
  Instantly see the price needed to exit your position without loss.

* 😨 **Fear & Greed Index**
  Market sentiment indicator for better decision-making.

* ⚠️ **Risk Assessment**
  Dynamic risk level based on volatility and performance.

* 🧠 **Smart Insights**
  AI-style commentary on your position and market conditions.

## 🛠 Tech Stack

* **Python**

## 📦 Installation (Local)

Clone the repository:

```bash
git clone https://github.com/yourusername/btc-advanced-dashboard.git
cd btc-advanced-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```
